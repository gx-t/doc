* SDL, SDL_ttf build build from sources, install
* Unlike brew install ... building from source enables SDL_GpuDevice

git clone --depth 1 https://github.com/libsdl-org/SDL.git
cd SDL
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make -j 8
sudo make install
git clone --depth 1 https://github.com/libsdl-org/SDL_ttf.git
cd SDL_ttf
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make -j 8
sudo make install
